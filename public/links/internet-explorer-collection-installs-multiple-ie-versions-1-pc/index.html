Internet Explorer Collection: Installs multiple IE versions on 1 PC

Internet Explorer Collection is an installer that allows you to install multiple versions of IE on 1 Windows instance. It includes the IE 1.0, 1.5, 2.01, 3.0, 4.01, 5.01, 5.5, 6.0, 7.0, and 8.0.

http://utilu.com/IECollection/
