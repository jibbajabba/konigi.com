Lorem Ipsum

Definition

Lorem Ipsum refers to the meaningless text used as placeholder or filler in visual design to suggest where text should be placed for layout. Other terms for this filler include Greeking, Placeholder Text, and Dummy Text. The text starting "Lorem ipsum dolor..." was traditionally a Latin passage by Cicero with words altered, added and removed that make it nonsensical in meaning and not proper Latin.
Source: Wikipedia

Text Generators

Traditional

  • Lipsum - Standard Lorem Ipsum that lets you set number of paragraphs, list items, etc. Also available as a Google Chrome Extension.
  • HTML Ipsum - HTML marked up standard Lorem Ipsum
  • Little Ipsum - Menubar app for Mac to copy greeked words, sentences or paragraphs as plain text or HTML.
  • Lorem Ipsum Me - Copy Blog or Essay length lorem ipsum passages to clipboard. Provides an API to use with curl or Ruby.
  • PakaPaka - Mac greeking app. Generate Greek, Classical Greek, German, Hebrew, Hindi, Old French, and of course, Bla- blah.

Non-Traditional

  • Corporate Ipsum Mac Widget and Google Chrome Extension - Business Bullshit generator.
  • Fillerati - Literary ipsum. Select an author and title, and choose an excerpt rendered in html or plain text.
  • Futureipsum - Futurama style filler, Google Chrome Extension
  • Gangsta Ipsum - Gangsta style ipsum.
  • Samuel L. Ipsum - Samuel L. Jackson style ipsum.
