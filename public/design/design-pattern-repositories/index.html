Design Pattern Repositories

About

This is a list of resources for design pattern research. The list contains books and web sites with formal design pattern libraries that include categorized patterns with problem statement and description of solution.

If you're looking for examples of design solutions in the wild, you may be interested in seeing the User Interface Galleries page.

Books

  • Design Of Sites, by Douglas K. Van Duyne, James A. Landay, Jason I. Hong.
  • Designing Interfaces, by Jennifer Tidwell.
  • Designing Web Interfaces: Principles and Patterns for Rich Interactions, by Theresa Neil. Book Blog has updated examples.
  • Designing Social Interfaces, by Erin Malone and Christian Crumlish.
  • Mobile Design Pattern Gallery, by Theresa Neil.
