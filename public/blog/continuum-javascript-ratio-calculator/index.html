Continuum Javascript Ratio Calculator

Continuum's Javascript Ratio Calculator is a handy tool you can use to calculate object dimensions based on a ratio you provide. You can do this easily in a graphics tool like Photoshop by setting an image size and resizing, but this can be useful when you just need to pull those dimensions out when planning a project layout.

http://www.continuum2.com/js_ratio.php
