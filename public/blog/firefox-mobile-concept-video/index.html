Firefox Mobile Concept Video

Firefox Mobile Concept Video from Aza Raskin on Vimeo.

Aza Raskin demonstrates a touch interface concept for a Firefox Mobile browser. The mocked up prototype includes features including spatial browsing of browser tabs, and touch gestures for summoning application menus.
